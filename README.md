# CS152Project
Pokedex & Tools in Python using Pokemon API(https://pokeapi.co/) and SQL to handle the back end.

HOW TO USE:
1. Open by running the file main.py.
2. Select the tool you want to use
   a. Pokedex allows you to see Pokemon information and search add pokemon that aren't present.
   b. Teambuilder allows you to build teams of 6 Pokemon by selecting and saving their information. Export the information into a txt file that can be used on https://pokemonshowdown.com/.
   c. Damage calculator allows you to select an attacker, a defender, and a move, to calculate the damage based on game logic.
